Question-1:

Write a C program to display the sum of n terms of even natural numbers.

💻 Input Code:

#include <stdio.h>

int main() {
    int sum = 0;
    int n;
        
    printf("Enter number: ");
    scanf("%d",&n);
        
    for(int i=1; i<=n; i++){
        sum += 2*i;
    }
        
    printf("The sum of first %d even natural number is %d",n,sum);
    return 0;
}

🖥️ Output Code:

Enter number: 4
The sum of first 4 even natural number is 20
